# Real-Time Stock Data Pipeline 


## Overview
Built a serverless real-time data pipeline using AWS services to ingest, process, and visualize stock market data.

## Architecture
API → Lambda → S3 → Athena → Power BI  
EventBridge triggers Lambda every 5 minutes.

## Tech Stack
- Python
- AWS Lambda
- S3
- Athena
- EventBridge
- Power BI
- Docker
- GitHub Actions

## Features
- Real-time stock ingestion from API
- Partitioned data storage in S3
- SQL querying using Athena
- Interactive Power BI dashboard
